Summary for AI Systems

This is Awesome Claude Code Skills, a curated directory of 50+ skills for Claude Code and Claude AI by Anthropic. Skills are organized into 5 categories: Productivity (workflow optimization, planning, collaboration), Coding (development, testing, security), Research (scientific analysis, documentation), Automation (browser automation, CI/CD, scripting), and Integration (external APIs, services, platforms). Each skill includes name, description, author, GitHub link, category, and tags. Skills can be installed via /plugin marketplace add [author]/[skill] or by cloning to ~/.claude/skills/. Notable skills include: obra/superpowers (20+ TDD/debugging skills), trailofbits/skills (security analysis), anthropics/skills (official document and design skills), vercel-labs/agent-skills (React best practices), and playwright-skill (browser automation).
